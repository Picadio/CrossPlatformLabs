# CrossPlatformLabs

## Installation
1. Clone project
2. Install **dotnet sdk**
3. Build project

## Usage

### Build
```bash
dotnet build Build.proj -p:Solution=X -t:Build
```

### Test
```bash
dotnet build Build.proj -p:Solution=X -t:Test
```

### Run
```bash
dotnet build Build.proj -p:Solution=X -t:Run
```

*Where X - Lab1, Lab2 or Lab3*
